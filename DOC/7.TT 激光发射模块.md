# **TT 激光发射模块**
## **简介**
<img src="IMG/TT 激光发射模块.png" title="TT 手势识别模块" width="50%" align="right">  

```
    TT激光发射模块使用5mw小型激光头,重量轻巧安全无害.采用PH2.0接口可直接连接TT IO扩展板驱动.
    这款产品重量轻巧不影响飞行器的正常飞行,无需任何工具就可以完成安装与拆卸.






```
---
## **产品参数**
<ul>
<li>驱动电路:Mos电路</li>
<li>功率:5MW</li>
<li>激光波长:650nm</li>
<li>光源:点状光源</li>
<li>工作电压:5V</li>
<li>接口:PH2.0-2P</li>
</ul>
